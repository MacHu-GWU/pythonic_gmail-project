Install
------------------------------------------------------------------------------

``pythonic_gmail`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install pythonic-gmail

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade pythonic-gmail
